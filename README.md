# Power BI HR Analytics Dashboard

## Project Overview
This project presents an interactive HR Analytics dashboard developed using Power BI to analyze employee headcount, salary distribution, age demographics, gender diversity, and performance ratings.

## Tools & Technologies
- Power BI
- Excel / CSV Dataset
- Data Modeling & DAX (basic measures)

## Dashboard Highlights
- Headcount analysis by department and gender
- Age distribution using bins
- Salary analysis (minimum, maximum, average)
- Performance rating distribution
- Time-based employee trend analysis

## Key Insights
- Website and Procurement departments have the highest headcount
- Majority of employees fall in the 25–35 age group
- Average salary varies significantly across departments
- Performance ratings are evenly distributed across categories
